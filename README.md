# MP4 → AVI Converter

Convertidor de videos usando FFmpeg WASM.

## Características

- MP4 → AVI
- Funciona en navegador
- Compatible con GitHub Pages
- Descarga ZIP
- Drag & Drop
- Consola en tiempo real

## Publicar en GitHub Pages

1. Subir archivos al repositorio
2. Ir a Settings → Pages
3. Deploy from branch
4. Seleccionar main
5. Abrir la URL generada
